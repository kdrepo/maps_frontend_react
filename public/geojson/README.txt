Place GeoJSON files from https://github.com/datta07/INDIAN-SHAPEFILES here.

Expected files:
- india_states.geojson
- states/<STATE_NAME>.geojson (one file per state)

Example:
public/geojson/india_states.geojson
public/geojson/states/Karnataka.geojson

After downloading, simplify with mapshaper and keep only the needed properties:
https://mapshaper.org
